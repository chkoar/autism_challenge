Coding framework

The challenge will be carried out on the RAMP platform. RAMP provides the necessary tools to collaborate and develop innovative solutions to solve data science problems.

For this challenge, the starting kit to start hacking is available: [HERE]. It provides a minimum working example which can be expanded to more advanced solutions.

What is this challenge?

Some nice introduction about the problem that we try to solve.

When?

This challenge is made of 2 phases:

  • On June 8, a RAMP session will take place in which any data scientists from all walk of like are invited.
  • On June 11, neuroscientists taking part of the Brainhack network are kindly invited to join a second collaborative RAMP session.

Where?

The event will be held in Paris at the Brain and Spine Institute (ICM) within one of the largest European hospitals in Europe: the Pitié-Salpêtrière hospital.
